Concepto de GIT

git es un repositorio de versiones distribuido, un clon local del proyecto el cual permite trabajar sin tener acceso a internet o de una forma remota con facibilidad, sirve para gestionar las versiones por las que va pasando el codigo de los proyectos

PRINCIPALES COMANDOS DE GIT

  • GIT INIT: Este comando funciona para inicializra un proyecto con git
  • GIT ADD: Permite agregar cambios al area de reparacion
  • GIT CLONE: Permite descartar el codigo fuente existente de un repositorio remoto
  • GIT BRANCH: Permite crear nuevas ramas o eliminarlas
  • GIT STATUS: Nos brinda informacion necesaria de una rama
  • GIT ADD: Permite crear o modificar los archivos
  • GIT COMMIT: Permite guardar los cambios
  • GIT PUSH: Envia los commits al repositorio remoto
